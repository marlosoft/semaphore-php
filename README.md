# Semaphore SMS

### Unofficial [Semaphore](https://www.semaphore.co/) library for sending SMS

### Installation
```bash
composer require marlosoft/semaphore-php
```

### Current Features
- [x] Send SMS
- [x] Send Bulk SMS
- [x] Send OTP
- [x] Get all messages
- [x] Get message by ID
- [x] Retrieve account
- [x] Retrieve account transactions
- [x] Retrieve account sender names
- [x] Retrieve account users
